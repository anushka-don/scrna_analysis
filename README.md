# scanpy  
single cell RNA seq Analysis using scanpy.  
It performes QC, preprocessing, Normalization, feature selection, PCA, UMAP and tSNE clustering, Integration, Marker gene analysis, automatic and manual annotation. Pseudotime analysis (PAGA) Cell Proportion analysis (scCODA) Cell-Cell Interaction analysis (CellPhoneDB)  

Reference: Zhang, S., Fang, W., Zhou, S. et al. Single cell transcriptomic analyses implicate an immunosuppressive tumor microenvironment in pancreatic cancer liver metastasis. Nat Commun 14, 5123 (2023). https://doi.org/10.1038/s41467-023-40727-7
